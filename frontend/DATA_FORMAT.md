# DATA_FORMAT.md

## Формат JSON-мотива

Мотив — это небольшой фрагмент схемы, который пользователь может перенести на рабочую область.

Пример:

```json
{
  "char": "B",
  "width": 18,
  "height": 21,
  "cells": [
    [0, 0, 1, 1],
    [0, 1, 0, 0]
  ]
}
```

## Поля мотива

### `char`

Необязательное поле. Может использоваться для буквенных мотивов.

```json
"char": "B"
```

### `name`

Необязательное поле. Может использоваться для декоративных мотивов.

```json
"name": "flower"
```

### `width`

Ширина мотива в клетках.

### `height`

Высота мотива в клетках.

### `cells`

Двумерный массив клеток.

Допустимые значения:

```text
0    пустая клетка
1    заполненная клетка
null прозрачная/невидимая клетка
```

## Семантика `null`

`null` в мотиве означает прозрачную клетку.

Правила:

- в предпросмотре `null` выглядит как пустая клетка;
- при переносе на рабочую область `null` ничего не меняет;
- `null` не должен попадать в рабочую область.

## Формат index.json

Браузер не может сам получить список файлов из папки, поэтому нужен файл-манифест.

Путь:

```text
public/patterns/index.json
```

Формат:

```json
{
  "patterns": [
    "B.json",
    "flower.json"
  ]
}
```

Приложение загружает каждый файл относительно `/patterns/`.

## Формат сохраненной рабочей области

Файл, который пользователь скачивает через “Сохранить JSON”:

```json
{
  "version": 1,
  "type": "filet-crochet-canvas",
  "width": 100,
  "height": 100,
  "cells": [
    [0, 1, 0],
    [1, 0, 1]
  ]
}
```

## Поля сохраненной рабочей области

### `version`

Версия формата. Для MVP использовать:

```json
"version": 1
```

### `type`

Тип файла. Для MVP использовать:

```json
"type": "filet-crochet-canvas"
```

### `width`

Ширина рабочей области.

### `height`

Высота рабочей области.

### `cells`

Двумерный массив только из `0` и `1`.

`null` в сохраненной рабочей области не допускается.

## Валидация мотива

Мотив валиден, если:

- `width` — положительное число;
- `height` — положительное число;
- `cells` — массив;
- количество строк равно `height`;
- длина каждой строки равна `width`;
- каждая клетка равна `0`, `1` или `null`.

## Валидация рабочей области

Сохраненная рабочая область валидна, если:

- `version` равен `1`;
- `type` равен `filet-crochet-canvas`;
- `width` — положительное число;
- `height` — положительное число;
- `cells` — массив;
- количество строк равно `height`;
- длина каждой строки равна `width`;
- каждая клетка равна `0` или `1`.
