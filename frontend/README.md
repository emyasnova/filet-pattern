# Filet Crochet Pattern Editor

Frontend-приложение для создания схем филейного вязания из клеток и готовых JSON-мотивов.

## MVP

Пользователь может:

- редактировать рабочую область из клеток;
- загружать готовые мотивы из JSON-файлов;
- поворачивать и отражать мотивы;
- перетаскивать мотивы на рабочую область;
- заполнять и очищать клетки кликом;
- экспортировать итоговую схему в PNG;
- сохранять рабочую область в JSON;
- загружать рабочую область из JSON;
- очищать рабочую область.

## Стек

- React
- Vite
- TypeScript
- CSS Modules или обычный CSS по решению реализации

Backend в MVP не используется.

## Запуск

После создания проекта:

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```

## Данные мотивов

Готовые мотивы лежат в публичной папке:

```text
public/
  patterns/
    index.json
    B.json
    flower.json
    ...
```

`index.json` содержит список JSON-файлов мотивов:

```json
{
  "patterns": [
    "B.json",
    "flower.json"
  ]
}
```

Приложение загружает `index.json`, затем загружает каждый перечисленный JSON-файл.

## Основной интерфейс

Экран делится на две зоны:

```text
┌───────────────────────────────┬───────────────────────┐
│ Рабочая область                │ Панель мотивов         │
│                               │                       │
│ Сетка клеток                   │ Мотив 1               │
│                               │ [rotate] [flip H] ... │
│                               │                       │
│                               │ Мотив 2               │
│                               │ ...                   │
└───────────────────────────────┴───────────────────────┘
```

Левая область:

- рабочая сетка;
- клик по клетке переключает `0 ↔ 1`;
- поддерживает drop мотивов;
- показывает preview вставки во время перетаскивания.

Правая область:

- вертикальный список карточек мотивов;
- у панели есть вертикальный скролл;
- каждая карточка имеет кнопки:
  - повернуть на 90°;
  - отразить по горизонтали;
  - отразить по вертикали.

## Экспорт

Кнопка “Сгенерировать изображение” скачивает PNG с итоговой схемой.

Изображение должно содержать:

- белый фон;
- черные заполненные клетки;
- тонкую серую сетку;
- более яркие разделители по 10 клеток;
- текст с размерами схемы в клетках.
