# TESTING_CHECKLIST.md

## Проверка загрузки мотивов

- [ ] При старте приложение загружает `/patterns/index.json`.
- [ ] Все файлы из `index.json` загружаются.
- [ ] Невалидный мотив не ломает все приложение.
- [ ] В правой панели видны карточки мотивов.

## Проверка отображения рабочей области

- [ ] По умолчанию создается сетка 100 × 100.
- [ ] Пустые клетки белые.
- [ ] Заполненные клетки черные.
- [ ] Сетка видна.
- [ ] Каждые 10 клеток выделены более ярко.
- [ ] Рабочая область остается удобной при большом размере.

## Проверка клика по клетке

- [ ] Клик по пустой клетке заполняет ее.
- [ ] Клик по заполненной клетке очищает ее.
- [ ] Несколько быстрых кликов работают корректно.

## Проверка изменения размера

- [ ] Увеличение ширины сохраняет старое содержимое.
- [ ] Увеличение высоты сохраняет старое содержимое.
- [ ] Новые клетки после увеличения пустые.
- [ ] Уменьшение ширины обрезает лишние столбцы.
- [ ] Уменьшение высоты обрезает лишние строки.
- [ ] После изменения размера можно продолжать редактирование.

## Проверка трансформаций мотивов

- [ ] Поворот на 90° меняет ширину и высоту местами.
- [ ] Горизонтальное отражение работает корректно.
- [ ] Вертикальное отражение работает корректно.
- [ ] Трансформация одной карточки не меняет другие карточки.
- [ ] После нескольких трансформаций мотив остается корректным.

## Проверка drag-and-drop

- [ ] Мотив можно начать перетаскивать из правой панели.
- [ ] При наведении на рабочую область появляется preview.
- [ ] Preview соответствует текущему transformed-состоянию мотива.
- [ ] Drop вставляет мотив в нужное место.
- [ ] `1` из мотива записывает заполненную клетку.
- [ ] `0` из мотива очищает клетку.
- [ ] `null` не меняет клетку рабочей области.
- [ ] При вставке за границу лишняя часть обрезается.
- [ ] Drop на край рабочей области не вызывает ошибку.

## Проверка сохранения и загрузки JSON

- [ ] Кнопка “Сохранить JSON” скачивает файл.
- [ ] Файл содержит `version`, `type`, `width`, `height`, `cells`.
- [ ] В `cells` сохранены только `0` и `1`.
- [ ] Загруженный файл восстанавливает размер рабочей области.
- [ ] Загруженный файл восстанавливает клетки.
- [ ] Невалидный файл показывает ошибку и не ломает текущее состояние.

## Проверка очистки

- [ ] Кнопка “Очистить” запрашивает подтверждение.
- [ ] При подтверждении все клетки становятся пустыми.
- [ ] Размер рабочей области не меняется.
- [ ] При отмене ничего не меняется.

## Проверка PNG

- [ ] Кнопка “Сгенерировать изображение” скачивает PNG.
- [ ] PNG имеет белый фон.
- [ ] Заполненные клетки черные.
- [ ] Пустые клетки белые.
- [ ] Тонкая серая сетка видна.
- [ ] Разделители каждые 10 клеток заметнее обычной сетки.
- [ ] На PNG есть текст с шириной и высотой схемы.
- [ ] На PNG нет лишних подписей.
- [ ] PNG корректен после изменения размера рабочей области.
- [ ] PNG корректен после загрузки JSON.
